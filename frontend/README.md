
## Sobre o projeto

Projeto desenvovidocom intuito de fazer listagem de bebidas. O site foi desenvolvido com ReactJS e React Bootstrap. 

## Pré-requisitos

- [Node.js](https://nodejs.org/en/) (na versão 16 ou superior)
- [NPM](https://www.npmjs.com/)

## Como executar?
```bash
npm install
npm run dev
```
